Apple iOS 11.3

  • Vendor: Apple
  • Version: iOS 11.3
  • Release Date: March 29, 2018
  • Emojis: 2613

iOS 11.3 was a minor update to the iOS emoji set. The main change in this release is the modification of four emojis used in Apple's separate Animoji feature. First announced in January 2018, this release was finalized in March 2018.

This emoji release also applied to tvOS 11.3, macOS 10.13.4 and watchOS 4.3. More about this update.
